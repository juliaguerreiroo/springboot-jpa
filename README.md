# Web Services com Spring Boot e JPA
[![NPM](https://img.shields.io/npm/l/react)](https://github.com/juliaguerreiroo/springboot-jpa/blob/main/LICENSE) 

# Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aprofundar os conhecimentos em desenvolvimento de Web Services utilizando Java com Spring Boot.

API REST desenvolvida para demonstrar na prática a aplicação de conceitos de persistência de dados com JPA e arquitetura em camadas utilizando Spring Boot.

# Tecnologias utilizadas
## Back end
- Java
- Spring Boot
- JPA / Hibernate
- Maven
- Banco de dados em memória H2

# Como executar o projeto

## Back end
Pré-requisitos: Java 17 ou superior

```bash
# clonar repositório
git clone https://github.com/juliaguerreiroo/springboot-jpa.git

# acessar a pasta do projeto
cd springboot-jpa

# executar o projeto
./mvnw spring-boot:run

```

## Testando a API
Você pode testar os endpoints utilizando:

* Postman

Realizando requisições como:

GET (buscar dados)
POST (criar)
PUT (atualizar)
DELETE (remover)
